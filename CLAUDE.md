# Проект: Персональный сайт врача — Екатерина Полуян

## Обзор
Персональный одностраничный сайт гинеколога-репродуктолога Екатерины Полуян.
Премиальный дизайн в стиле editorial / luxury wellness.

## Стек
- **Один HTML-файл** (`index.html`) — всё inline (CSS + JS + контент)
- **Шрифты**: Google Fonts — `Lora` (serif) + `Montserrat` (sans-serif)
- **Цветовая палитра**: тиффани (#0abfbc), белый, серый, мягкие оттенки
- **CSS-переменные** заданы в `:root`
- **Адаптивность**: множество `@media` брейкпоинтов, включая ручные mobile-fix

## Структура страницы (секции)
1. **Nav** — sticky навигация с CTA-кнопкой «Записаться»
2. **Hero** (`#top`) — имя врача, специализация, статистика (21 год, 15 000+ пациентов, 1 000+ операций), CTA
3. **About** (`#about`) — биография, принципы работы, цитата
4. **Cases** (`#cases`) — 3 клинических кейса (карточки)
5. **Reviews** (`#reviews`) — отзывы пациентов (карусель)
6. **Posts** (`#posts`) — публикации (3 карточки, ссылки на Telegram/Дзен)
7. **Contact** (`#contact`) — блок для сотрудничества с коллегами
8. **Booking** (`#booking`) — форма записи (очный / онлайн), отправка в Telegram/Formspree
9. **Channels** — ссылки на каналы
10. **Footer** — контакты, навигация, копирайт

## SEO (уже реализовано)
- `<title>` с ключевыми запросами
- `<meta name="description">` и `<meta name="keywords">`
- Open Graph теги (og:title, og:description, og:locale, og:site_name)
- Schema.org JSON-LD разметка `Physician` (специализации, клиника, адрес, услуги)
- `<link rel="canonical">`
- H1 содержит «Репродуктолог и гинеколог в Москве»
- H2 оптимизированы под целевые запросы

## Форма записи
- Два формата: «Очный приём» / «Онлайн-консультация»
- Поля: имя, контакт, описание ситуации
- Отправка: Telegram Bot API → Formspree (fallback)
- **Токены-заглушки**: `YOUR_BOT_TOKEN`, `YOUR_CHAT_ID`, `YOUR_FORMSPREE_ID` — требуют замены
