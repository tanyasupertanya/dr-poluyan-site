# Екатерина Полуян — персональный сайт врача

Персональный сайт гинеколога-репродуктолога. Одностраничный, премиальный дизайн.

## Быстрый старт

Просто откройте `index.html` в браузере. Никаких зависимостей и сборки не требуется.

## Настройка формы записи

В файле `index.html` найдите и замените:
- `YOUR_BOT_TOKEN` → токен вашего Telegram-бота
- `YOUR_CHAT_ID` → ID чата для получения заявок
- `YOUR_FORMSPREE_ID` → ID формы на Formspree (резервный канал)
