# Чеклист запуска dr-poluyan.ru

Этап 1 — всё, что нужно сделать **до** публикации сайта и индексации в поисковиках. Делайте по порядку — многие пункты зависят друг от друга.

---

## 1. Домены и 301-редиректы (reg.ru)

Основной домен: **dr-poluyan.ru** (canonical, на него всё ссылается).
Зеркала: **ekaterina-poluyan.ru** и **dr-poluyan.com** — должны 301-редиректить на `https://dr-poluyan.ru`.

### На reg.ru:
1. Зайти в панель управления → «Мои домены».
2. Для `dr-poluyan.ru`:
   - Привязать к хостингу (или Cloudflare Pages / Netlify / GitHub Pages — уточнить, где хостим).
   - Подключить SSL-сертификат (Let's Encrypt бесплатно).
3. Для `ekaterina-poluyan.ru`:
   - В разделе «Домен → Переадресация» установить 301-редирект на `https://dr-poluyan.ru`.
   - Редирект должен сохранять путь: `/bla` → `https://dr-poluyan.ru/bla`.
4. Для `dr-poluyan.com` — то же самое.

> ⚠️ **Не ставьте 302 (временный) — только 301 (постоянный).** Иначе Яндекс и Google не склеят зеркала и будут считать контент дублем.

## 3. Счётчики аналитики

### Яндекс.Метрика
1. [metrika.yandex.ru](https://metrika.yandex.ru) → «Добавить счётчик».
2. Адрес сайта: `dr-poluyan.ru`. Часовой пояс: Москва. Включить: карта кликов, вебвизор, точный показатель отказа.
3. Скопировать **номер счётчика** (8 цифр) → вставить вместо `YANDEX_METRIKA_ID` в `index.html` (2 места).

### Google Analytics 4
1. [analytics.google.com](https://analytics.google.com) → создать ресурс.
2. Тип данных: Web. URL: `https://dr-poluyan.ru`.
3. Скопировать **Measurement ID** (вид `G-XXXXXXXXXX`) → заменить в `index.html` (2 места).

## 4. Яндекс.Вебмастер

1. [webmaster.yandex.ru](https://webmaster.yandex.ru) → «Добавить сайт» → `https://dr-poluyan.ru`.
2. Подтверждение прав: выбрать **метатег** → скопировать content → вставить вместо `YANDEX_VERIFICATION_CODE` в `<head>` `index.html`.
3. После публикации сайта нажать «Проверить».
4. **Настройки → Главное зеркало:** указать `https://dr-poluyan.ru`. ekaterina-poluyan.ru и dr-poluyan.com добавить как отдельные сайты и тоже привязать главным зеркалом `dr-poluyan.ru` — это ускорит склейку.
5. **Настройки → Регион сайта:** указать «Москва».
6. **Индексирование → Файлы Sitemap:** добавить `https://dr-poluyan.ru/sitemap.xml`.
7. **Диагностика → Переобход страниц:** добавить главную.

## 5. Google Search Console

1. [search.google.com/search-console](https://search.google.com/search-console) → «Добавить ресурс» → «URL-префикс» → `https://dr-poluyan.ru`.
2. Подтверждение: HTML-тег → вставить вместо `GOOGLE_VERIFICATION_CODE` в `<head>`.
3. **Sitemaps** → добавить `sitemap.xml`.
4. **Settings → Change of address** — НЕ нужно (это новый сайт, не переезд).

## 10. После публикации — проверка

- [ ] Открыть `https://dr-poluyan.ru` — работает ли SSL (замочек).
- [ ] Открыть `https://ekaterina-poluyan.ru` → должен редиректнуть на dr-poluyan.ru.
- [ ] Открыть `https://dr-poluyan.com` → аналогично.
- [ ] [search.google.com/test/rich-results](https://search.google.com/test/rich-results) → проверить JSON-LD (должен распознать Physician + MedicalClinic).
- [ ] [validator.schema.org](https://validator.schema.org) — то же самое.
- [ ] [pagespeed.web.dev](https://pagespeed.web.dev) — скорость (целевой Performance 85+).
- [ ] Проверить, что Я.Метрика видит посещения (вкладка «Посещаемость»).
- [ ] В Я.Вебмастере через 1–3 дня — статус «Подтверждено», индексация началась.
